- 留言板系统

## 前端相关：

- https://element-plus.org/zh-CN/guide/design.html
- https://cn.vuejs.org/guide/quick-start.html

## 功能点梳理

- 用户模块
    - 注册
    - 登录
    - 个人中心
    - 管理员新增用户
    - 修改个人资料
    - 分页查询用户列表
- 标签模块
    - 新增标签
    - 分页查询标签
    - 查询所有标签
- 分类
    - 新增分类
    - 分页查询分类
    - 查询所有分类
- 留言模块
    - 新增留言
        - 留言关联分类
        - 留言关联标签
    - 我发表的留言列表
    - 留言详情
    - 留言列表
    - 评论留言
- 文件模块
    - 文件上传
    - 访问已上传的文件
